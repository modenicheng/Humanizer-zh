# 质量评分

> **触发条件：** 用户要求评分时，或改写完成后自评时。
>
> **使用方式：** 对改写文本逐项打分，给出总分与改进建议。

---

## 评分维度（总分 50）

| 维度 | 评估标准 | 得分 |
| ------ | ---------- | ------ |
| **直接性** | 绕圈子了吗？直说事实，还是宣告意义？<br>10 分：开门见山；1 分：铺垫成瘾 | /10 |
| **节奏** | 句子长短有变化吗？读起来顺不顺？<br>10 分：错落有致；1 分：整齐如排队 | /10 |
| **信任度** | 过度解释了吗？<br>10 分：该说的说，该省的就省；1 分：唯恐读者不懂 | /10 |
| **真实性** | 像真人说话吗？<br>10 分：隔壁老王聊天；1 分：新闻稿机器人 | /10 |
| **精炼度** | 还有废话可删吗？<br>10 分：删一字都不行；1 分：一半可删 | /10 |
| **总分** |  | **/50** |

---

## 中文特有扣分项

以下问题源自余光中《翻译与文学的常态与变态》，扣分时单独列出：

| 问题类型 | 扣分规则 | 示例 |
|---------|---------|------|
| 弱动词+抽象名词 | 每处 -1 | 「进行研究」→ 应改为「研究」 |
| 「被」字滥用 | 每处 -1 | 「被选为」→ 应改为「当选」 |
| 「的」字密度过高（>5%，即每 20 个字一个「的」） | 整体 -3 | 「美丽的宁静的夜晚」 |
| 抽象名词泛滥（X性、X度、X力） | 每处 -1 | 「知名度」「可行性」 |
| 连接词冗余（和/与/及/以及） | 每处 -0.5 | 「温暖与舒适」 |
| 填充式介词（关于/有关/对于） | 每处 -0.5 | 「关于这个问题」 |

> **注意：** 中文扣分不改变总分上限（仍为 /50），单独列出以识别被西化破坏的中文习惯。

---

## 评分标准

| 分数段 | 等级 | 说明 |
|--------|------|------|
| 45-50 | 优秀 | 听不出 AI 味，像真人说话 |
| 35-44 | 良好 | 大体过关，尚有几处可改进 |
| 25-34 | 一般 | AI 痕迹明显，须重新打磨 |
| 低于 25 | 差 | 建议重写 |

---

## 输出格式

按以下格式输出：

```text
【评分结果】总分：42/50

| 维度 | 得分 | 说明 |
|------|------|------|
| 直接性 | 9/10 | 开头直接，结尾略绕 |
| 节奏 | 8/10 | 整体顺畅，中间有两句等长 |
| 信任度 | 9/10 | 克制，不过度解释 |
| 真实性 | 8/10 | 较自然，仍有一处"此外"可用可删 |
| 精炼度 | 8/10 | "进行了分析"可改为"分析了" |

中文扣分：弱动词 ×2 (-2)、「的」密度 6% (-3)
主要问题：第 3 段两个「进行」结构、第 5 段「的」字偏密。
```
